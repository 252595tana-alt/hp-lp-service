# STAR INVADERS（スマートフォン専用シューティングゲーム）

縦画面のスマートフォンブラウザで遊べる、スペースインベーダー風のシンプルなシューティングゲームです。
HTML / CSS / JavaScript（Canvas 2D）のみで構成されており、外部ライブラリには依存していません。

## 1. ファイル構成

```
game/
├── index.html   … ゲーム画面のHTML構造（スタート/ステージ切替/ゲームオーバー/クリア画面を含む）
├── style.css    … レトロ宇宙風のスタイル、レスポンシブレイアウト、操作ボタンのデザイン
├── game.js      … ゲームロジック本体（設定値・入力・描画・当たり判定・ステージ管理）
├── assets/      … 画像素材を置くディレクトリ（ボス画像を配置する場所）
└── README.md    … このファイル
```

## 2. 起動方法

このゲームは静的ファイルのみで動作するため、特別なビルド作業は不要です。

### そのままファイルを開く
`index.html` をスマートフォンまたはPCのブラウザで直接開くだけで動作します。

### ローカルサーバーで確認する場合（推奨）
一部のブラウザではローカルファイルを直接開くと画像読み込み等で制限が出ることがあるため、
簡易サーバーを立てての確認を推奨します。

```bash
cd game
python3 -m http.server 8000
# ブラウザで http://localhost:8000/ を開く
```

Node.js がある場合は以下でも起動できます。

```bash
npx serve game
```

## 3. スマートフォンで確認する方法

1. 上記のローカルサーバーを起動したPCと、確認したいスマートフォンを同じWi-Fiに接続する
2. PCのIPアドレス（例: `192.168.1.10`）を確認する
3. スマートフォンのブラウザで `http://192.168.1.10:8000/` を開く
4. 画面を縦向きにした状態でプレイする（横向きにすると回転案内が表示されます）

Webサーバーにデプロイする場合は、`game` フォルダの中身一式をそのままアップロードし、
`index.html` にアクセスするURLを開けば遊べます。

## 4. ボス画像の差し替え方法

`game.js` 内の `CONFIG.boss.imagePath` がボス画像のパスです。

```js
// game.js 内
boss: {
  ...
  imagePath: 'assets/boss.png', // ← このパスを変更するだけで画像を差し替え可能
},
```

差し替え手順：

1. 使用したいボス画像（背景透過PNG推奨）を `game/assets/boss.png` として配置する
   （別ファイル名にしたい場合は `imagePath` の値も合わせて変更する）
2. 画像は縦横比を保ったまま、ボスの表示枠（`CONFIG.boss.width` / `height`）内に収まるよう自動調整されます
3. 画像が存在しない・読み込みに失敗した場合は、自動的に仮図形（六角形のシルエット）で表示されるため、
   画像を用意する前でも開発・確認が可能です

### スタート画面から写真を選んで差し替える（コード編集不要）

ファイルを直接置き換える以外に、スタート画面の「▼ ボス画像を設定」から人物写真を選ぶだけで
ボス画像を差し替えることもできます。

- 選んだ写真の顔部分を自動でトリミングして正方形のボス画像を生成します
- Shape Detection API（`FaceDetector`）に対応した端末では実際に顔を検出してトリミングします
- 非対応の端末（iOS Safariなど）では、写真の上部中央付近を顔とみなして自動トリミングします
- 処理はすべて端末内（ブラウザ内）で完結し、画像が外部サーバーへ送信されることはありません
- 選んだ画像は端末の `localStorage` に保存され、次回アクセス時も引き継がれます
- 「リセット」ボタンでデフォルトの `assets/boss.png`（未設置の場合は仮図形）に戻せます

## 5. 操作方法

- 画面下部の「◀」「▶」ボタンを押している間、自機が左右に移動します
- 「FIRE」ボタンで弾を発射します（長押しで連射、連射速度には上限があります）
- 移動と発射は同時に操作できます（マルチタッチ対応）

## 6. 主な仕様

- ステージ1（通常敵15体）→ ボス戦 → ゲームクリア
- プレイヤーの初期ライフは3、被弾後は一定時間の無敵状態になります
- ボスの初期HPは30、被弾のたびに1減少し、画面上部にHPゲージを表示します
- スコア：通常敵100点、強い敵200点、ボス撃破3,000点
- Canvasは端末のPixel Ratioに合わせて描画されるため、高解像度端末でもぼやけません

## 7. コード構成について

`game.js` は以下のセクションに分かれています。

1. `CONFIG` … 数値設定（速度・HP・スコアなど）をまとめた設定オブジェクト
2. ユーティリティ関数（`clamp` / `aabbHit` など）
3. `AudioEngine` … 効果音（Web Audio APIによる合成音、外部音声ファイル不要）
4. 描画クラス群（`Star` / `Bullet` / `Player` / `Enemy` / `Boss` / `Explosion`）
5. `Input` / `bindHoldButton` … Pointer Eventsによる入力管理（マルチタッチ対応）
6. `BossImage` … ボス画像の読み込みとフォールバック処理
7. `Game` … ステージ管理・当たり判定・HUD更新・画面遷移を行う本体オブジェクト
